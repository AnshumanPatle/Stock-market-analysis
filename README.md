# Stock Market Analysis

## Overview
Analysis of stock market performance across multiple sectors to identify
price trends, volatility patterns, and sector-wise returns.

## Tools Used
- Excel — data preparation and cleaning
- Tableau — interactive visualizations and dashboards
- Power BI — performance dashboard

## Key Analysis
- Sector-wise stock performance comparison
- Price movement and volatility trends
- Volume analysis over time
- Top performing and underperforming stocks
